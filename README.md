# Logistics Platform

A modern logistics and shipment tracking website built with Vue.js, Node.js/Express, and PostgreSQL.

## Features

- User Authentication (Login/Signup)
- Shipment Tracking Dashboard
- Real-time Order Status Updates
- Admin Panel
- Payment Integration

## Tech Stack

- **Frontend**: Vue 3
- **Backend**: Node.js/Express
- **Database**: PostgreSQL
- **Architecture**: Monorepo

## Project Structure

```
logistics-platform/
├── frontend/          # Vue.js application
├── backend/           # Express.js server
├── docs/              # Documentation
└── README.md
```

## Getting Started

### Prerequisites
- Node.js 16+
- PostgreSQL 12+
- npm or yarn

### Installation

#### Backend Setup
```bash
cd backend
npm install
npm run dev
```

#### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
